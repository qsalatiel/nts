# Node Typescript Starter

Features

- Node 20+
- Express
- Winston Logging
- Docker first
- Eslint + Prettier
- Bruno HTTP docs

### How to run development mode

**Requirements: **

- Docker installed

1. Clone the repository using degit: `npx degit https://github.com/qsalatiel/nts MyProjectName`
2. `cd MyProjectName`
3. `npm install`
4. `npm run up:dev`
